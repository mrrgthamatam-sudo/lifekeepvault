# 🛡️ EternalVault — Install on Your PC

## Step 1: Install Node.js (one-time only)

1. Go to **https://nodejs.org**
2. Click the big green **LTS** button
3. Run the downloaded installer → click Next through everything
4. **Restart your PC**

To verify: open Command Prompt and type `node --version`

## Step 2: Install EternalVault

1. Download/copy this entire project folder to your PC
2. **Double-click `BUILD-EXE.bat`**
3. Press any key when prompted
4. Wait 3-5 minutes (first run downloads Electron ~85 MB)
5. **Done!** The app launches automatically

## What Gets Installed

| Item | Location |
|------|----------|
| App files | `C:\Users\YourName\AppData\Local\EternalVault\` |
| Desktop shortcut | `EternalVault` icon on your Desktop |
| Start Menu | Search "EternalVault" in Start Menu |

## How to Launch

- **Double-click** the `EternalVault` icon on your Desktop
- Or search **EternalVault** in the Start Menu
- Or run `C:\Users\YourName\AppData\Local\EternalVault\EternalVault.exe`

## How to Uninstall

1. Delete the folder: `C:\Users\YourName\AppData\Local\EternalVault`
2. Delete the Desktop shortcut: right-click → Delete
3. That's it — no registry entries, no leftover files

## Privacy

- **100% Offline** — works without internet
- **No cloud** — zero data leaves your device
- **No tracking** — no analytics, no telemetry
- **Local storage** — all data in your browser/app storage
